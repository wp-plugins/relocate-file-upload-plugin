== Description ==

Ever wish you could move the File Upload iframe section on the default WordPress Post and Page editors? Now you can. This plugin will make the File Upload box conform to the functionality similar to the Custom Fields, Trackacks. You will be able to collapse the File Upload section to free some screen relesate or even re-order the section using the drag and drop ability. 
[Plugin Homepage](http://www.codehooligans.com/2007/04/07/update-to-wordpress-relocate-file-upload-plug-in/ "Relocate File Upload Plugin")

== Installation ==

1. Upload `relocate_file_upload` folder and contaiined file `relocate_file_upload.php` the your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Write -> Post or Write -> Page. 

[Plugin Homepage](http://www.codehooligans.com/2007/04/07/update-to-wordpress-relocate-file-upload-plug-in/ "Relocate File Upload Plugin")

== Frequently Asked Questions ==

= Will this plugin function if I disable JavaScript =

No. Actually most of the Admin section of WordPress will not function is you turn off JavaScript.
